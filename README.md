
# University - Multi Pages Website

The University Multi-page website is built using HTML, CSS, and JavaScript. It is a comprehensive website consisting of multiple pages that provide information about the university. The website incorporates HTML for structuring the content, CSS for styling and layout, and JavaScript for interactivity and enhanced functionality. It offers a user-friendly interface, responsive design, and engaging features to provide an optimal browsing experience. Visitors can navigate through various pages to access information about courses, Blogs, About, campus facilities, and more. The combination of HTML, CSS, and JavaScript ensures a visually appealing and interactive website.
